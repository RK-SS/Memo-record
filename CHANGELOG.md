# 更新日志

所有重要的项目变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-12-25

### 新增
- 🎉 首次发布
- ✨ 分组管理功能，支持创建、编辑、删除分组
- 📋 卡片式笔记，支持标题和内容编辑
- 🎨 自定义分组颜色，8 种预设颜色可选
- 🔍 全局搜索功能，支持搜索标题和内容
- 📝 Markdown 支持，集成代码高亮（支持 18 种常用语言）
- 🌓 主题切换功能（浅色/深色/跟随系统）
- 💾 自动备份功能，每次数据变更时自动创建备份
- 📦 手动备份功能，支持立即创建备份点
- 🔄 数据恢复功能，从备份历史中恢复数据
- 📤 多格式导出：
  - 简化 JSON（仅标题和内容）
  - Markdown 文档
  - 完整备份（包含所有元数据）
- 📥 数据导入功能：
  - 支持完整备份格式
  - 支持简化 JSON 格式
  - 合并/覆盖两种导入模式
- 🖱️ 拖拽排序，分组和卡片都支持拖拽调整顺序
- 🔐 密码保护，首次启动生成随机密码
- ⚙️ 备份配置，可设置保留备份数量（5/10/20/50）
- ℹ️ 关于页面，显示软件信息和作者信息
- 📊 仪表盘，显示统计信息和最近编辑

### 优化
- 🚀 打包体积优化，JS 文件减少 70%（1.25MB → 394KB）
- 📦 单文件打包，生成 portable 可执行文件
- 🎨 深色模式适配，所有页面完美支持
- 💾 北京时间显示，备份文件名和历史记录使用本地时间
- 🔧 代码高亮优化，按需引入常用语言，减小体积

### 修复
- 🐛 修复导入功能的序列化问题
- 🐛 修复导出下拉菜单事件冒泡
- 🐛 修复初始密码显示问题
- 🐛 修复 TypeScript 类型警告

### 技术栈
- Electron 25.9.8
- Vue 3.3.0
- TypeScript 5.0.0
- Vite 4.5.14
- Pinia 2.1.0
- Vue Router 4.2.0
- marked 17.0.1
- highlight.js 11.11.1
